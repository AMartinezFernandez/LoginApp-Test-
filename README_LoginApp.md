LoginApp – Proyecto Android en Kotlin

Aplicación Android de ejemplo para autenticación de usuarios con Firebase Authentication.
Incluye inicio de sesión, registro, paso de datos entre pantallas y validaciones.

Características
- Login con email y contraseña
- Registro de usuarios
- Validación de campos
- Paso de datos entre Activities
- RecyclerView para mostrar productos
- Firebase integrado (opcional)

 Tecnologías utilizadas
- Kotlin
- Android Studio
- Firebase

---
Proyecto desarrollado como práctica para DAM2 – Acceso a Datos.
